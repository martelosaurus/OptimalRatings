Prerequisites:
1. All scripts are written in Python3. Required packages include <matplotlib>, <numpy>, and <scipy>.
2. Download Dow Jones Industrial Average, 1885-02-16 through 20210-09-03, from MeasuringWorth (https://www.measuringworth.com/datasets/DJA/index.php). Save as "DJA.csv" in a sub-directory called "data". 

In a terminal, run "python3 driver.py".
